# 亚马逊团队 - 财务分析师记忆

## 定价模型

### FBA利润计算公式
```
毛利润 = 售价 - 产品成本 - FBA费用 - 头程费用
净利润 = 毛利润 - 广告费 - 退款损失 - 其他费用
毛利率 = 毛利润 / 售价
```

### 盈亏平衡ACOS
```
盈亏ACOS = (售价 - 产品成本 - FBA费用) / 售价
```
