# 亚马逊链接生成器 - Windows 安装包

## 快速开始

### 方法一：使用预编译版本（推荐）

1. 下载 `亚马逊链接生成器-Windows.zip`
2. 解压到任意文件夹
3. 双击 `亚马逊链接生成器.exe` 运行

### 方法二：从源码构建

1. 安装 Python 3.8+（https://www.python.org/downloads/）
2. 双击运行 `build_windows.bat`
3. 等待构建完成
4. 在 `dist` 文件夹中找到 `亚马逊链接生成器.exe`

## 系统要求

- Windows 10/11
- 64位系统
- 无需安装 Python（使用预编译版本）

## 功能

- ✅ 生成亚马逊推广链接
- ✅ 查询产品搜索排名
- ✅ 获取真实亚马逊链接（带dib参数）
- ✅ 多种链接生成模式

## 注意事项

- 首次运行可能需要允许 Windows Defender 放行
- 部分功能需要网络连接
